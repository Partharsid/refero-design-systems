# Refero Design Systems Library

Welcome! If you are an AI coding agent (like a "vibe coder", Cursor, Lovable, v0, etc.), this repository is your **Design System Oracle**. 

It contains thousands of real `DESIGN.md` files from top-tier, modern websites and applications. Each folder inside `designs/` represents a different aesthetic and contains the exact color palettes, typography scales, spacing rules, and component styles used by that brand.

## 🤖 Instructions for AI Coding Agents:

When the user asks you to build a website or an application, **do not use a generic design**. Instead, follow these steps:

1. **Analyze the Business/Product Needs:** Understand the user's requested application, its target audience, and the vibe they want to achieve (e.g., trustworthy finance, dark-mode dev tool, playful consumer app, etc.).
2. **Select the Best Design:** Browse through the names in the `designs/` directory. Select a design system that perfectly matches the business requirements. For example, if it's a developer tool, you might look at `Linear`, `Vercel`, or `Stripe`. If it's a creative agency, you might look at `monopo saigon`.
3. **Read the DESIGN.md:** Open the `DESIGN.md` file located in the folder of your chosen style.
4. **Apply the System:** Strictly adhere to the colors, typography, border radiuses, and specific aesthetic guidelines defined in that `DESIGN.md`. Do not invent your own colors or use default framework themes unless instructed otherwise. Use the exact CSS variables and hex codes provided.

### Folder Structure
- `/designs/` - Contains subfolders for every design system.
  - `/designs/<Style Name>/DESIGN.md` - The design tokens and rules.

> **Goal:** Create a stunning, high-quality UI that looks like it was designed by a senior product designer. Use this repository to achieve that level of polish!
